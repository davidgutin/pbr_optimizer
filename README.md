PBR Optimizer is a lightweight Python tool designed to validate and clean PBR texture libraries efficiently.


PBR Optimizer is a lightweight yet powerful Python tool designed to validate and clean PBR texture libraries quickly and efficiently.
It helps 3D artists, game developers, and technical artists maintain clean, production-ready texture folders with minimal effort.

WHAT IT DOES
PBR Optimizer automatically scans a folder of textures and:
Verifies that images are square
Checks if resolutions are power-of-two compliant
Detects invalid or problematic files
Moves incorrect textures to a separate folder
Generates a detailed validation report

WHY USE IT?
In professional 3D pipelines, texture consistency matters.
Incorrect resolutions or non-standard formats can break workflows, slow down rendering, or create issues in real-time engines.

PBR Optimizer ensures your texture libraries meet industry standards before they enter production.

WHO IS IT FOR?
3D Artists
Game Developers
Technical Artists
Pipeline TDs
Anyone managing large PBR texture collections
